Shooting Game
==============
* Toy project
* Multiplayer
* HTML5 (Canvas, WebSockets)

How to play
---------

prerequisites:

* python3
* tornado (`pip3 install tornado`)

fetch the code: (`git clone `Username/Repository)


run the server:

```
cd python
python3 launch.py
```

play: browse http://localhost:8888/

ask your friend to join
